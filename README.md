# Java DSA - Linked List

## 📖 Overview

This repository contains Java implementations of Linked List data structures along with important coding interview problems.

The goal of this project is to strengthen Data Structures and Algorithms concepts for coding interviews and product-based company placements.

## 🚀 Topics Covered

### Linked List Basics

- Singly Linked List
- Doubly Linked List
- Circular Linked List

### Operations

- Insert at Beginning
- Insert at End
- Insert at Position
- Delete First Node
- Delete Last Node
- Delete by Position
- Search Element
- Update Node
- Display List
- Count Nodes


## 📚 Interview Problems

- Reverse Linked List
- Find Middle Node
- Detect Cycle
- Remove Nth Node From End
- Merge Two Sorted Lists
- Check Palindrome Linked List
- Find Intersection Point
- Delete Node Without Head
- Remove Duplicates
- Rotate Linked List
